## Usage

**Basic**

```
docker run -it --rm dockette/hashicorp lavant deploy app.hcl
```

**Gitlab CI**

If you whatever reason don't want to use [Vercel for Gitlab](https://vercel.com/gitlab).

```yml
stages:
 - deploy

deploy:
  stage: deploy
  image: dockette/hashicorp
  script:
    - nomad job run app.nomad
```
